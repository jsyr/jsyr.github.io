Notable Open Source Contributions

BlackAdder Network Node System

An alternative fully functional linux kernel network stack architecture for the Internet. Contributed to several subsystems.

The design is documented in this Paper: G. Parisis, D. Trossen and D. Syrivelis, Implementation and evaluation of an information-centric network, IFIP Networking 2013 (Presented Core System of the Pursuit project which won 2013 FIA award)

IPsec support for the MIT Click modular router project

In this work i have added IPsec support to the MIT Click Modular Router Project. The implementation and how-to can be found in Click-Wiki. This work was a result of my collaboration with Princeton University.

Volatile Software Suspend for uClinux kernel

In this work, i have developed a custom, first stage loader assisted, suspend-resume scheme for uClinux kernel. The system suspends execution by storing the H/w state in volatile memory. The overall process is extremely fast and the kernel is able to resume execution on adifferent H/w platform. On reconfigurable hardware platforms this is indeed useful. This work is backed up by a USENIX Annual Technical Conference Publication.

Porting the Blob bootloader and linux kernel to NMI uEngine platform

This work is now outdated because the hardware platform is outdated. It was done back in 2003 for the IST project 2WEAR. Patches have been integrated to the 2.4 series mainline Arm linux kernel.
